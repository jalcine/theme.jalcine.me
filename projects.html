NoPhucks (Given)

So as a way for me to hack and improve a bit of the web technologies I know of, I built nophucks. At the moment, it's a self-contained application (meaning all of its actions and logic are stored in a few client-side files) using Backbone, jQuery and Compass. The app's my way of understanding functional programming via JavaScript (a la Underscore), and also gives me the chance to experiment with CSS3 + HTML5 features like flex-box and LocalStorage. Take a look at the source code or check it out live on my site.
