  • I live over in Oakland (on PDT / PST) but can work remotely by about +/- 2 hours.
  • By trade, I'm a full-stack web developer working with Python, PHP and Go.
  • This is a free service so we won't be working on paid work. Let's work together to improve open tools and software.
